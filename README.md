# Integrating Tonic with gRPC Web

This repo demonstrates the integration of Tonic with gRPC Web, as outlined in the article available at <https://rust-on-nails.com/docs/api/grpc-web/>.

## Requirements

- Rust compiller
- `pnpm`
- `protoc`

## Quickstart

```bash
make run-client &
make run-server
```

Then

```bash
open http://localhost:4173/
```
